# AutoTreeChop
A auto tree chopping plugin for milkteamc.  
Discord: https://discord.gg/pyNS5xAvMs   
## Permission and commands
usage: `/autotreechop`  
description: Toggle auto tree chop.  
permission: `autotreechop.use`  
## Config
`config.yml`
```yml
messages:
  enabled: "§a已開啟自動砍樹。"
  disabled: "§c已關閉自動砍樹。"
  no-permission: "§c你沒有權限使用此指令。"
```
